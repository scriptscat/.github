ScriptCat，是一个可以执行用户脚本的浏览器扩展，相比油猴脚本拥有后台脚本运行的框架与更多的API。

我们致力于推广与发展更强更好用的用户脚本，希望每个人都能因此受益，我们不仅拥有完善的学习资料，并且拥有一个活跃的社区，你可以通过下面的链接进入。
